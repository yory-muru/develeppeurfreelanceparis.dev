│   └── main.js                  # JavaScript interactif
├── assets/
│   ├── favicon.svg              # Favicon
│   └── logo.svg                 # Logo
├── freelances/
│   └── index.html               # Page pour freelances
├── clients/
│   └── index.html               # Page pour clients
├── blog/
│   ├── index.html               # Liste des articles

## Optimisations SEO

- Balises meta title et description uniques par page
- URLs propres avec mots-clés
- Hiérarchie H1 > H2 > H3 respectée
- Schema.org (Organization, FAQ, Article)
- Open Graph et Twitter Cards
- Canonical URLs
- Maillage interne complet
- Pages noindex pour login/register
- Sitemap-ready structure

## Déploiement GitHub Pages

1. Créer un repository GitHub
2. Pousser le code
3. Activer GitHub Pages dans Settings > Pages
4. Configurer le domaine personnalisé
